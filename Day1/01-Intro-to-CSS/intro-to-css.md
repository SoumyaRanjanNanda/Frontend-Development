# CSS (Cascading Style Sheets)

## Definition

**CSS (Cascading Style Sheets)** is a style sheet language used to **control the appearance and layout of web pages** written in HTML.

It describes **how HTML elements should be displayed** on screen, paper, or other media.

---

## Short Explanation

* **HTML → Structure of a webpage**
* **CSS → Design and styling of a webpage**

Example:
HTML creates a **heading**, CSS decides its **color, size, position, and font**.

---

## Example Code

```html
<!DOCTYPE html>
<html>
<head>
<style>
h1{
  color: blue;
  font-size: 40px;
}
</style>
</head>

<body>

<h1>Hello World</h1>

</body>
</html>
```

### Output

The **Hello World text becomes blue and large** because of CSS.

---

# Key Points (Impactful)

1. CSS stands for **Cascading Style Sheets**.
2. It is used to **style and design HTML elements**.
3. CSS controls **colors, fonts, spacing, layout, and animations**.
4. It separates **content (HTML)** from **design (CSS)**.
5. CSS makes websites **beautiful, responsive, and user-friendly**.
